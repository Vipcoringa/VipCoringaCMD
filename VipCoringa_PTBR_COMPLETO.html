Gerenciamento Wi-Fi

netsh wlan show profiles

Use quando: Precisar ver todas redes Wi-Fi salvas
Saída: Lista de redes no formato "Perfil de todos os usuários: NOMEDAREDE"

netsh wlan show profile name="NETWORK" key=clear

Use quando: Quiser ver a senha de uma rede específica
Dica: Substitua NETWORK pelo nome exato da rede. A senha aparece em "Conteúdo da chave"

GERENCIAMENTO WI-FI

Ferramentas de Wi-Fi

Comandos poderosos para gerenciar e solucionar problemas de conexão Wi-Fi

Comandos Essenciais de Wi-Fi

1. Ver redes salvas

netsh wlan show profiles

Lista todas as redes Wi-Fi salvas no computador.

2. Ver detalhes da rede (incluindo senha)

netsh wlan show profile name="NETWORK_NAME" key=clear

Substitua "NOME_DA_REDE" pelo nome exato da rede. A senha aparecerá em "Conteúdo da Chave".

3. Ver conexão atual

netsh wlan show interfaces

Mostra a rede Wi-Fi atual, força do sinal, SSID, BSSID, etc.

Operações Avançadas de Wi-Fi

4. Exportar perfil de rede para backup

netsh wlan export profile name="NETWORK_NAME" folder="C:\Backup" key=clear

Cria um arquivo .xml com o perfil da rede e senha.

5. Importar perfil de rede

netsh wlan add profile filename="C:\Backup\PROFILE.xml"

Reinstala um perfil de rede previamente salvo.

6. Apagar uma rede salva

netsh wlan delete profile name="NETWORK_NAME"

Remove uma rede Wi-Fi dos perfis salvos.

DIAGNÓSTICO DE REDE

Comandos de Rede

Ferramentas essenciais para diagnosticar problemas de rede

Configuração de IP

ipconfig /all

Mostra todas as informações de IP (IPv4, gateway, máscara).

ipconfig /release

Libera o endereço IP atual.

ipconfig /renew

Solicita um novo IP do roteador.

ipconfig /flushdns

Limpa o cache DNS para resolver erros de navegação.

Teste de Conexão

ping google.com

Testa a conectividade com a internet.

tracert google.com

Exibe o caminho (roteadores) até o destino.

Análise de Rede

netstat -ano

Mostra todas as conexões ativas e suas portas.

arp -a

Exibe a tabela ARP (endereços MAC da rede local).

nslookup google.com

Consulta o IP de um domínio.

getmac

Mostra o endereço MAC do seu PC.

Ferramentas Avançadas

route print

Exibe a tabela de roteamento.

net use

Mostra os discos de rede mapeados.

ADMINISTRAÇÃO DO SISTEMA

Comandos de Administração do Sistema

Comandos poderosos para gerenciar e solucionar problemas no seu Windows

Informações do Sistema

systeminfo

Exibe informações completas do sistema, incluindo SO, hardware e mais.

Gerenciamento de Processos

tasklist

Lista todos os processos em execução.

taskkill /F /IM process.exe

Encerra forçadamente um processo pelo nome.

Controles de Energia

shutdown /r /t 0

Reinicia o PC imediatamente.

shutdown /s /t 0

Desliga o PC imediatamente.

Reparo do Sistema

sfc /scannow

Verifica e repara arquivos corrompidos do sistema.

chkdsk /f

Verifica e corrige erros no disco.

Gerenciamento de Usuários

whoami

Mostra o nome do usuário atual.

net user

Lista os usuários do sistema.

net user username /add

Cria um novo usuário.

Programas e Drivers

wmic product get name,version

Lista os programas instalados.

driverquery

Lista os drivers instalados.

msinfo32

Abre a interface gráfica de informações do sistema.
